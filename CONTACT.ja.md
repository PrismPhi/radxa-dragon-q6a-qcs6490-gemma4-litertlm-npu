# 連絡先

技術的な質問、再現報告、不具合報告は、まず `docs/reproducibility_checklist.ja.md`
を確認した上でGitHub Issuesを使ってください。

雑談や軽いコメントについては、maintainerのGitHub profile linksを参照してください。
Twitter/XなどのURLは変わる可能性があるため、このrepository内には直接固定しません。

以下は投稿しないでください。

- credentialを含むprivate log
- SSH接続情報
- model access token
- publicではないQualcomm SDK download link
- proprietary runtime library
