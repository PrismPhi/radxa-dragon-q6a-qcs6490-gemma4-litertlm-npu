# Contact

For technical questions, reproducibility reports, or bugs, please use GitHub
Issues after checking `docs/reproducibility_checklist.md`.

For informal comments, see the maintainer's GitHub profile links. This keeps
the repository free of personal contact URLs that may change over time.

Please do not post:

- private logs containing credentials,
- SSH details,
- model access tokens,
- Qualcomm SDK download links that are not public,
- proprietary runtime libraries.
