# プロジェクトの由来

このrepositoryは、ローカルで進めたQ6A/Gemma4 LiteRT-LM NPU実験を、公開向けに
整理したものです。

このプロジェクトと文書の作成には、OpenAI Codexをかなり使用しています。
公開内容についてはmaintainerが確認・選別・編集し、責任を持ちます。

## これが意味すること

- 文書、script、patch整理にはAI支援が含まれる可能性があります。
- 実際の動作は、利用者側のQ6A環境、patched LiteRT-LM binary、model artifact、
  Qualcomm runtime libraryに依存します。
- Codexを使ったことは、OpenAI、Google、Qualcomm、Radxaのvendor supportを意味しません。
- 派生成果物を公開する前に、各自の環境でcommandとlicenseを確認してください。
