# Project Provenance

This repository is a cleaned public version of a local Q6A/Gemma4 LiteRT-LM NPU
experiment.

The project and documentation were prepared with substantial assistance from
OpenAI Codex. The maintainer reviewed, selected, edited, and is responsible for
the published contents.

## What This Means

- The repository may contain AI-assisted wording, scripts, and patch
  organization.
- The actual runtime behavior still depends on the user's Q6A environment,
  patched LiteRT-LM binary, model artifact, and Qualcomm runtime libraries.
- AI assistance does not imply vendor support from OpenAI, Google, Qualcomm, or
  Radxa.
- Please verify commands and licenses in your own environment before publishing
  derived artifacts.
